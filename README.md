# 🎡 Jet Lag · Race to Athens — Spinner

A themed spinning wheel for your **Modern Euro Period 8** game. One self-contained
file (`index.html`) — no installs, no build step, no dependencies.

## ✨ What it does

- A beautiful canvas wheel styled to match the *Jet Lag* slideshow (cream + gold
  frame, dusty-pink script, typewriter accents).
- **Built-in presets** straight from the game:
  - 🌍 **Countries** — all 20 countries you can travel to
  - 👥 **Teams** — the 4 teams (editable)
  - 🎲 **Difficulty** — Easy / Medium / Hard
  - 💀 **Curses** — split into **Easy / Medium / Hard** so you only ever spin a
    curse from the correct tier (the winner's full effect is shown in the result popup)
  - ✍️ **Custom** — type your own list (auto-saved in your browser)
- Edit any list live — one option per line.
- **Remove winner after spin** toggle for picking with no repeats.
- Tick + chime sound (toggleable), confetti, keyboard support (press **Space** to spin),
  and a mobile-friendly layout.

## ▶️ Use it right now (no hosting needed)

Just **double-click `index.html`** — it opens in any browser and works fully offline.
To share with your group, send them the file or zip the folder.

## 🌐 Put it online for free (so everyone gets one link)

### Option A — GitHub Pages (recommended, permanent link)
1. Create a free account at [github.com](https://github.com).
2. Make a new **public** repository (e.g. `jetlag-spinner`).
3. Upload `index.html` (and this README) — drag-and-drop in the GitHub web uploader.
4. Go to **Settings → Pages**, set **Source: Deploy from a branch**, branch **main**, folder **/ (root)**, **Save**.
5. Wait ~1 minute. Your link will be:
   `https://YOUR-USERNAME.github.io/jetlag-spinner/`

### Option B — Netlify Drop (fastest, ~30 seconds)
1. Go to [app.netlify.com/drop](https://app.netlify.com/drop).
2. Drag this whole **folder** onto the page.
3. You instantly get a public link to share. (Free Netlify account keeps it permanent.)

### Option C — Cloudflare Pages / Vercel
Both have free tiers and accept a folder with a single `index.html`. Point them at
this folder and deploy — no configuration required.

> All options are free. Since everything lives in `index.html`, there is nothing to
> configure — whatever host you pick just serves that one file.

## 🛠️ Customizing

- **Quick edits:** open the site, pick a preset, and edit the text box (one option
  per line). Custom lists are remembered on that device.
- **Permanent edits:** open `index.html` in any text editor and edit the `PRESETS`
  block near the top of the `<script>` section.
